Icon 图标 #

@shinho-yt/sh-design-icons-vue 提供了一套常用的图标集合。

安装 #

使用包管理器 #

# 选择一个你喜欢的包管理器

# NPM
$ npm install @shinho-yt/sh-design-icons-vue@latest
# Yarn
$ yarn add @shinho-yt/sh-design-icons-vue@latest
# pnpm
$ pnpm install @shinho-yt/sh-design-icons-vue@latest

基础用法 #

<template>
  <div>
    <!-- 使用 sh-icon 为 SVG 图标提供属性 -->
    <sh-icon :size="size" :color="color">
      <UserFillIcon />
    </sh-icon>

    <!-- 或者独立使用它，不从父级获取属性 -->
    <AiIcon />
  </div>
</template>
