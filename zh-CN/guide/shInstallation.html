安装与快速上手 #

设置 registry #

npm set registry http://10.211.62.41:4873

安装组件库 #

通过 npm/yarn/pnpm 安装

npm install @shinho-yt/sh-design-plus or
yarn add @shinho-yt/sh-design-plus or
pnpm install @shinho-yt/sh-design-plus

在项目入口文件（一般是 main.js 或者 main.ts）注册组件库 #

安装 sh-design-plus，可同时使用 element-plus 组件及 sh-design-plus 扩充组件。 sh-design-plus 包中根据 设计侧提供的公司主题风格重置 element-plus 组件样式。

引入方式

import { createApp } from 'vue';
import ShDesignPlus from '@shinho-yt/sh-design-plus';
import '@shinho-yt/sh-design-plus/packages/theme/style/index.scss';

createApp(App)
  .use(shDesignPlus)
  .mount('#app');

由于组件全量导入可能会增加项目打包体积，因为可选择动态导入或进行配置文件的优化（如Tree Shaking）降低项目打包体积。


样式入口文件说明 #

默认提供两套主题色，分别为欣和红 与 品牌蓝， 引入方式参考如下：

  1. 欣和红主题色&EP样式风格重置请导入 import '@shinho-yt/sh-design-plus/packages/theme/style/index.scss';
  2. 品牌蓝主题色&EP样式风格重置请导入 import '@shinho-yt/sh-design-plus/packages/theme/style/index.scss';

如项目存在自定义主题风格，不需要自带的 element-plus 主题重置样式包， 方式参考如下：

  1. 欣和红主题色&EP样式风格重置请导入 import '@shinho-yt/sh-design-plus/packages/theme/style/shinho-theme.scss';
  2. 品牌蓝主题色&EP样式风格重置请导入 import '@shinho-yt/sh-design-plus/packages/theme/style/brand-theme.scss';

基于Vite引入 svg 组件库 #

vite.config.js

import { defineConfig } from 'vite';
import vue from '@vitejs/plugin-vue';

const { resolve } = require('path');
const {
  svgBuilder
} = require('@shinho-yt/sh-design-plus/packages/icons/svgBuilder');

// https://vitejs.dev/config/
export default defineConfig({
  plugins: [vue(), svgBuilder(resolve(__dirname, 'node_modules/@shinho-yt/sh-design-plus/packages/icons/src/svg/'))]
});
基于webpack引入 svg 组件库 #

vue.config.js

module.exports = {
  chainWebpack: config => {
    const svgRule = config.module.rule('svg');

    // 清除已有的所有 loader。
    // 如果你不这样做，接下来的 loader 会附加在该规则现有的 loader 之后。
    svgRule.uses.clear();

    // 添加要替换的 loader
    svgRule
      .use('svg-sprite-loader')
      .loader('svg-sprite-loader')
      .options({
        symbolId: '[name]'
      });
  }
};

main.ts

const requireAll = (requireContext: any) => requireContext.keys().map(requireContext);

const req = require.context('@shinho-yt/sh-design-plus/packages/icons/src/svg/', false, /\.svg$/);

requireAll(req);
