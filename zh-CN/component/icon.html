Icon 图标 #

@shinho-yt/sh-design-icons-vue 提供了一套常用的图标集合。

安装 #

使用包管理器 #

# 选择一个你喜欢的包管理器

# NPM
$ npm install @shinho-yt/sh-design-icons-vue@next
# Yarn
$ yarn add @shinho-yt/sh-design-icons-vue@next
# pnpm
$ pnpm install @shinho-yt/sh-design-icons-vue@next

基础用法 #

<template>
  <div>
    <!-- 使用 el-icon 为 SVG 图标提供属性 -->
    <el-icon :size="size" :color="color">
      <UserFillShIcon />
    </el-icon>

    <!-- 或者独立使用它，不从父级获取属性 -->
    <AiShIcon />

    <!-- 注意通过 :prefix-icon 前缀或者 :suffix-icon 插入时 需要进行一次执行-->
    <el-time-picker v-model="value1" placeholder="Arbitrary time" :prefix-icon="ClockCircleShIcon()"/>
  </div>
</template>
