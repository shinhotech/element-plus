此指南将帮助您从 Element UI 2.x 升级到 Element Plus。

Vue 3 迁移构建 #

使用 Element Plus 与 Vue 3 迁移构建时可能遇到一些问题。 有些组件依赖于 Vue 3 内部 API。 尝试将全局组件或将你项目中每个组件的版本调整为3 。
