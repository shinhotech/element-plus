简介 #

欢迎使用 ShDesignPlus。

ShDesignPlus 是一个简洁的前端 UI 框架，基于 vue3.0 开发，内部引入开源组件 ElementPlus，并对 Element Plus 组件进行导出，在此基础上扩充基础组件、业务组件等功能，灵活适配公司业务。框架使用 vite、vue3.0、typescript 搭建， 组件脱胎于欣和技术中台项目组。

安装 sh-design-plus，可同时使用 ElementPlus 组件及 ShDesignPlus 扩充组件。

建议配合tailwindcss一起使用。

基于 vue 的版本 #

基于 Vue 3.1.2 版本，如果低于该版本，请升级vue的版本，后续版本中，不确定是否随时更新 vue 版本。

基于 ElementPlus 的版本 #

基于 ElementPlus 2.5.3 版本，导出组件，如果低于该版本， 需升级使用到的 ElementPlus 组件功能。

基于 scss #

使用之前，请安装使用 dart-scss
